Right Way of Duplicating Array & Objects

In JavaScript, copying arrays and objects can be done in several ways:

  1. Shallow Copy
  2. Deep Copy

Ways to Copy Arrays

Shallow Copy:
  1. Using slice() Method
  2. Using the Spread Operator
  3. Using Array.from() Method
  4. Using the concat Method
Deep Copy:
  1. Using JSON.parse() and JSON.stringify()
  2. Using a Custom Recursive Function

Ways to Copy Objects

Shallow Copy:
  1. Using Object.assign()
  2. Using the Spread Operator
Deep Copy:
  1. Using JSON.parse() and JSON.stringify()
  2. Using a Custom Recursive Function
