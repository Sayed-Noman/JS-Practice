Scope

In JavaScript, scope refers to the accessibility of variables, objects, and functions in different parts of your code:

  • Global Scope
  • Function Scope
  • Block Scope
  • Lexical Scope
  • Local Scope
  • Script Scope
  • Module Scope
  • Immediately Invoked Function Expression (IIFE) Scope

Summary of All Scopes:

Global Scope: Accessible throughout the entire code.

Function Scope: Variables are limited to the function they are declared in.

Block Scope: Variables declared with let or const are confined to the nearest block.

Lexical Scope: Inner functions can access variables from their outer functions.

Local Scope: Similar to function/block scope; emphasizes limited visibility within functions or blocks.

Script Scope: Variables declared at the top level of a script file/module, accessible throughout that file/module.

Module Scope: Scoped to a module, not accessible globally unless exported.

IIFE Scope: Creates a new scope for encapsulation.
