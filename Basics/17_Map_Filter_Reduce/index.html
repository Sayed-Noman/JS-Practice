Map, Filter, Reduce Methode

Map:

  • Purpose: The map method creates a new array by applying a specified function to each element of an existing array.
  • Immutability: It does not modify the original array; instead, it returns a new array.
  • Common Use Case:Transforming data (e.g., doubling numbers, extracting properties from objects). Chaining with other array methods like filter and reduce

Filter:

  • Purpose: The filter() method is designed to create a new array that contains only the elements from the original array that meet a specified condition. It's a way to extract a subset of elements based on a predicate function.
  • Immutability: The filter() method does not alter the original array; instead, it returns a new array.
  • Common Use Cases: Filtering Numbers, Filtering Objects, Searching and Validating Data
